PDF Utility

PDF画像重ね書き出し

PDFの各ページをブラウザ内で表示し、PNG・JPG・WebP画像をページ上に重ねて位置やサイズを調整できます。 サーバー送信は行わず、このブラウザ内だけで合成してPNG保存できます。

PDFと画像データは外部サーバーへ送信されません。すべてブラウザ内で処理します。

ページごとに重ね画像の状態を保持します。ページを切り替えても編集内容は維持されます。

PDFアップロード

重ね画像

選択中ページに画像を追加します。複数画像の重ね合わせに対応しています。

ページ選択後に追加できます。画像はブラウザ内のみで保持します。

選択中画像の調整

キャンバス上でドラッグ・リサイズし、数値でも微調整できます。

書き出し

プレビューは編集用、保存時は高解像度で再描画してPNGを書き出します。

PDFを読み込むと、ここに作業状況を表示します。

ページ編集

キャンバス上で画像をつかんで移動できます。四隅と辺のハンドルで自由にサイズ変更できます。
