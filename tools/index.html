Tool Index

ツール一覧

売買・賃貸・ヒアリング・PDF関連など、実務で使いやすいツールをまとめています。GitHub Pages 上でそのまま使える静的ツールです。

売買諸費用計算ツール

購入時にかかる諸費用や初期費用の目安を確認できるツール

カテゴリ: 売買 / 諸費用

住宅ローン返済シミュレーター

借入額、金利、返済期間から毎月返済額を確認できるツール

カテゴリ: 売買 / 住宅ローン

借入可能額シミュレーター

年収や返済条件から借入可能額の目安を確認できるツール

カテゴリ: 売買 / 借入可能額

賃料から見る購入予算シミュレーター

現在の賃料から、無理のない購入予算の目安を確認できるツール

カテゴリ: 売買 / 資金計画

固定資産税・都市計画税 日割り精算計算ツール

引渡し時の固定資産税・都市計画税の精算額を、日数・日額・税目別内訳・計算根拠つきで確認できるツール

カテゴリ: 売買 / 税金精算

賃貸初期費用計算ツール

賃貸契約時にかかる初期費用を項目別に確認できるツール

カテゴリ: 賃貸 / 初期費用

賃料増額リスク診断

賃料増額請求のリスクや注意点を整理するためのツール

カテゴリ: 賃貸 / 賃料改定

原状回復費用の概算ツール

退去時の原状回復費用の目安と説明材料を整理できるツール

カテゴリ: 賃貸 / 原状回復

法人契約リスクチェックツール

法人契約時の確認項目や注意点を整理できるチェックツール

カテゴリ: 賃貸 / 法人契約

売主ヒアリングシート

売却相談時に必要な確認項目を整理しやすいヒアリングツール

カテゴリ: ヒアリング / 売主

賃貸ヒアリングフォーム

賃貸のお客様向けヒアリング項目を整理しやすいフォーム

カテゴリ: ヒアリング / 準備中

PDF画像変換

カテゴリ: PDF / 画像変換
