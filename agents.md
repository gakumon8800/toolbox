# AGENTS.md

## Purpose
This repository hosts static tools for real estate and rental operations on GitHub Pages.

## Stack
- Static HTML, CSS, and JavaScript only
- No backend
- No framework unless explicitly requested
- Must work on GitHub Pages with relative paths

## Repository rules
- Each tool lives in its own folder
- Default file structure:
  - index.html
  - styles.css
  - script.js
- Keep implementation simple and maintainable
- Match existing toolbox design and navigation patterns
- Update the tool index/listing when adding a new tool

## UX rules
- Mobile-first
- Clear labels in Japanese
- Practical business tone
- Prioritize readability and trust over flashy design
- Important warnings must be clearly visible

## Coding rules
- Avoid unnecessary dependencies
- No broken relative links
- No console errors
- Keep logic in readable constants/functions
- Use comments only where they improve maintenance

## Done means
- Tool page is reachable from toolbox index
- Files are separated cleanly
- UI works on mobile
- No obvious layout breakage
- Result text is natural Japanese
